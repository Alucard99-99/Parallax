Este projeto foi desenvolvido utilizando HTML com a implementação do efeito parallax, com o objetivo de criar uma experiência visual mais dinâmica e interativa para o usuário. O site apresenta diferentes seções com imagens de fundo que se movimentam em velocidades distintas durante a rolagem da página, gerando um efeito de profundidade.

A estrutura foi construída com foco na organização do conteúdo e no uso de elementos semânticos do HTML, garantindo clareza e acessibilidade. O efeito parallax foi aplicado para enriquecer a apresentação visual e demonstrar na prática conceitos modernos de desenvolvimento web.

Este projeto contribuiu para o aprimoramento das minhas habilidades na criação de páginas estruturadas e na aplicação de recursos visuais que melhoram a experiência do usuário.
